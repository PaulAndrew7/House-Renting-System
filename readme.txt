Steps to Run the code:
1. Run the HouseRenting.java file.
2. If you have not signed up, then do sign up once and sign in again.
3. Further it shows options such as,
	1.Rent House
	2.View House
	3.Booked Properties
	4.Exit
4. If you want to add property, then enter option 1 and proceed with further options.
5. if you want to view houses to book, then enter option 2 and proceed with further options to view properties.
6. Once you have booked a property, it can be viewed in Booked properties option by entering option 3.
7. Then enter option 4 to log out, if you wish to exit.
